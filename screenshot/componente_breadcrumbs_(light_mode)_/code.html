Breadcrumbs

Navigation component to indicate the current page's location within a hierarchy. Helps users understand their current position and navigate back to parent levels.

React / TSX 
import { Breadcrumb, BreadcrumbItem, BreadcrumbLink, BreadcrumbSeparator } from '@nova-ui/breadcrumb'

export default function App() {
  return (
    <Breadcrumb>
      <BreadcrumbItem>
        <BreadcrumbLink href='/'>Home</BreadcrumbLink>
      </BreadcrumbItem>

      <BreadcrumbSeparator />

      <BreadcrumbItem>
        <BreadcrumbLink href='/components'>Components</BreadcrumbLink>
      </BreadcrumbItem>

      <BreadcrumbSeparator />

      <BreadcrumbItem isCurrentPage>
        <BreadcrumbLink href='/breadcrumbs'>Breadcrumbs</BreadcrumbLink>
      </BreadcrumbItem>
    </Breadcrumb>
  )
}

API Reference

Prop Type Default Description
separator ReactNode '/' Custom separator element between items.
addHomeIcon boolean false If true, adds a home icon to the first item automatically.
itemsAfterCollapse number 1 Number of items to show after the ellipsis when collapsed.
itemsBeforeCollapse number 1 Number of items to show before the ellipsis when collapsed.
check

Do

  • Use breadcrumbs for hierarchically structured sites greater than two levels deep.
  • Place breadcrumbs at the top left of the page, above the page title.
close

Don't

  • Don't use breadcrumbs as the primary navigation method.
  • Don't include the current page link if the page title is already prominent.
